# GAN-vault

Progetto didattico del corso **Generative AI** presso il Politecnico, dedicato allo studio e all'implementazione di modelli generativi avanzati.

## Contenuti

Il corso copre le principali famiglie di modelli generativi:
- **Generative Adversarial Networks (GANs)**
- **Transformers e modelli autoregressive**
- **Variational Autoencoders (VAEs)**
- **Latent Diffusion Models**
- **Normalizing Flows**
- **Reinforcement Learning** e ML Agents

## Struttura

- `raw/` - Trascrizioni delle lezioni e slide del corso
- `notes/` - Note tematiche sui modelli generativi
- `wiki/` - Documentazione e risorse
- `skills/` - Script e utility per l'elaborazione
- `log_execution/` - Log delle esecuzioni dei laboratori

## Obbiettivi

Comprendere come funzionano i modelli generativi, implementarli da zero per i casi base e utilizzare piattaforme open-source per applicazioni avanzate.
